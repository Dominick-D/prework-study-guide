HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag defines metadata about an HTML document. Metadata is data (information) about data.

CSS

  • A "card" in web design refers to a container element that groups related content together, such as a product listing or a social media post. It is typically styled with a border, box-shadow, and padding to create a distinctive look.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The "width" property sets the width of an element, either in pixels, ems, or as a percentage of its parent element's width.
  • Width can also be set to "auto", which makes the element take up as much space as needed to fit its content without overflowing.
  • The "box-shadow" property adds a shadow effect to an element, creating the illusion of depth and separation from the background.
  • The "color" property sets the color of text inside an element, while the "background-color" property sets the color of the background behind the element.
  • Colors can be specified in a variety of ways, including named colors (such as "red" or "blue"), hex codes (such as "#000000" or "#FFFFFF"), or RGB/RGBA values (such as "rgb(255, 0, 0)" or "rgba(0, 0, 255, 0.5)").

Git

  • cd - change directory
  • code . - open VS Code
  • git - command must precede every Git statement
  • checkout - command is used to move the working branch to a new branch
  • -b - flag creates a new branch
  • git status - verify the working branch and see the status of your files
  • add - command adds modifications in the current working branch to the staging area
  • -A - flag indicates that we want to add all changes to the staging area
  • git commit - command with a commit message that describes what the commit contains
  • pull - use command to receive a branch's modifications into the local environment
